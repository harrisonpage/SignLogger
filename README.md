# SignLogger

Log sign placement to console: timestamp, user, coordinates, message.

```
[17:46:32 INFO]: [SignLogger] <harrisonpage> -6,69,8 KILLROY WAS HERE
```

# Building

```
mvn clean ; mvn package
```

# Install

```
cp target/SignLogger-1.0.0.jar ~/minecraft/plugins/
```

(change target dir accordingly)
